Create a Flow of dismiss events on the AutoCompleteTextView instance.

Note: Created flow keeps a strong reference to the AutoCompleteTextView instance until the coroutine that launched the flow collector is cancelled.

Example of usage:

autoCompleteTextView.dismisses()
    .onEach {
         // handle auto-complete text view dismiss event
    }
    .launchIn(uiScope)
